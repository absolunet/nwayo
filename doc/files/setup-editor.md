# PHPStorm 

#### EditorConfig
```
> Preferences 
> Plugins > Browse Repositories...
	- Search 'EditorConfig'
	- Install plugin
```

#### SCSS Lint
```
> Preferences 
> Plugins > Browse Repositories...
	- Search 'scss-lint'
	- Install plugin

[Restart]

> Preferences 
> Other settings > SCSS Lint
	- [x] Enable
	- SCSS Lint exe : /usr/local/bin/scss-lint
	- [x] Let scss-lint search for config up the folder tree
```

#### JSHint
```
> Preferences 
> Languages & Frameworks > Javaqscript > Code Quality Tools > JSHint 
	- [x] Enable
	- [x] Use config files
  - Version (latest)
  - Location: [x] Default
```
